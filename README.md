  MiniLMS — система управления курсами (мини-LMS)
MiniLMS — это лёгкая учебная платформа, которая позволяет управлять курсами, уроками, учениками и домашними заданиями. Проект создан на Django и подходит как пример учебного проекта или как основа для реальной LMS-системы.

  Возможности
Управление курсами
Создание, редактирование и удаление курсов
Привязка уроков к курсу
Список всех уроков курса

  Управление учениками
Добавление учеников
Привязка ученика к курсу
Просмотр прогресса ученика

  Уроки
CRUD-управление уроками
Поддержка текстового контента
Пометка урока как завершённого для ученика

  Домашние задания
Создание заданий к урокам
Статус выполнения: на проверке → принято / отклонено
Оценки

  Email-уведомления
Уведомление ученика о новом уроке
Уведомление о проверенном ДЗ

  Сертификаты
Автоматическая генерация PDF-сертификата после завершения курса

  Технологии
Технология	Использование
Django 5.x	основа проекта
HTML + Bootstrap 5	интерфейс
SQLite / PostgreSQL	база данных
WeasyPrint / ReportLab	генерация PDF
Django Email Backend	уведомления
